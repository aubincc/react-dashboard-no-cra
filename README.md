# React 18 without create-react-app

## Description

This is simply React 18, without `create-react-app`.

## Install

`git clone https://github.com/aubincc/react18-no-cra.git`

`npm install`

## Mount locally

`npm run dev:start`

## Build for production

`npm run dev:build`

## Mount the production build

`npm run test:serve`
